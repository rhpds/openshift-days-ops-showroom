= Welcome

== OpenShift Days - Ops Track

Welcome to the OpenShift Operations workshop. This hands-on lab covers platform setup, Day 2 operations, and advanced capabilities for OpenShift administrators and platform engineers.

== Your Environment

You have access to an OpenShift 4.x cluster with:

* 3 control plane nodes
* 2+ worker nodes
* Cluster admin privileges

== Getting Started

Use the terminal and web console tabs to interact with your cluster. You're logged in as a cluster admin.

Verify your access:

[source,bash,role="execute"]
----
oc whoami
----

[source,bash,role="execute"]
----
oc get nodes
----

When ready, proceed to the first module in the navigation.
